# Nextcloud

[Nextcloud](https://nextcloud.com/) is a self-hosted cloud solution.
Despite being really feature rich I only use it as a Google drive/Dropbox alternative.

## Apps

- [Nextcloud Office](https://apps.nextcloud.com/apps/richdocumentscode) - Connects to a Collabora Online server and handle office files.
- [OpenID Connect user backend](https://apps.nextcloud.com/apps/user_oidc) - OIDC handling. Allow me to use [Authentik](../authentik)
