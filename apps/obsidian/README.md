# Obsidian

> [!WARNING]
> Obsidian is not open source. I've tried a few note taking apps and always landed back to Obsidian.

[Obsidian](https://obsidian.md/) is a cross-device note taking / second brain app (and much more than that).
The reason I talk about it there is because a lot of its component are open source (plugins, livesync, etc.).

## Theme

[Minimal](https://minimal.guide/home) with light theme Ayu

## Plugins

### Theming

Basically this [config](https://minimal.guide/guides/macos-config)

- [Minimal Theme Settings](https://github.com/kepano/obsidian-minimal-settings)
- [Hider](https://github.com/kepano/obsidian-hider)
- [Style Settings](https://github.com/obsidian-community/obsidian-style-settings)

### Utility

- [Excalidraw](https://github.com/zsviczian/obsidian-excalidraw-plugin)
- [Self-hosted LiveSync](https://github.com/vrtmrz/obsidian-livesync)
- [Calendar](https://github.com/liamcain/obsidian-calendar-plugin)

## Syncing between device

A lot of options are available for this, but I end up using [LiveSync](https://github.com/vrtmrz/obsidian-livesync) and despite the cumbersome setup I'm quite happy with it.
